# Jogo de Cartas do Yu-Gi-Oh - Scissor, Paper, Rock
>*Desafio da DIO - Criando um jogo de Cartas do Yu-Gi-Oh! Com JS e CSS*

</br>
<h2>Como funciona o jogo</h2>
<p style="text-indent: 20px;">O jogo se baseia no jogo de cartas e no jogo do jankenpon <i>(connhecido também por pedra, papel e tesoura)</i>. Seu inimigo é o computador, que lança aleatoriamente as cartas, com isso você pode ganhar, empatar ou perder, facilmente.</p>
<p style="text-indent: 20px;">Ademais, a lógica deste jogo é o mesmo que jankenpon, por exemplo, se o inimigo lança pedra e você, a tesoura, logo quem vence é seu inimigo, pois pedra quebra tesoura.</p>
